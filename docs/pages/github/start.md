# 快速开始

## 域名规则

| 仓库名称 | 默认域名 |
|---------|---------|
| `<username>.github.io` | `https://<username>.github.io/` |
| 其他名称 | `https://<username>.github.io/<repository>/` |

## 步骤

1. 创建仓库，推送静态文件
2. **Settings** → **Pages**
3. Source 选择分支和目录（`/ (root)` 或 `/docs`）
4. 点击 **Save**

## 入口文件优先级

1. `index.html`
2. `index.md`
3. `README.md`

## 部署状态

在仓库 **Actions** 标签页查看部署状态，通常 1-2 分钟生效。
